# Me👤

一个怀抱着热忱之心不甘于屈服于命运的大学生，此博客的搭建者。梦想是夏日里铺上凉席，手抱西瓜吹着风扇，然后看着电视里的奥特曼嘿嘿嘿，这种场景从小到大只有过一次来着。还有就是目前正在钻研设计方面的知识，为了成为一名设计师正在努力中… 但是我感觉自己的动手能力好差啊，每次在脑海里有了想法，打开 ps 的时候一下子就不知道怎么弄了 ，这是一个很令人头疼的问题，或许是软件不够熟练亦或是想法不完整不成熟。这些问题我都在改善中… 不过完事开头难，只要不放弃终会找到路！
